Настройки CryptoBota:
1) Создание токена CryptoBot /wallet -> Назад -> Crypto Pay -> Создать приложение -> API токен
2) В настройках приложения жмем "Безопасность" и включаем createCheck и transfer для автоматической оплаты и создания чеков
3) Создаем счет на оплату в CryptoBot /invoices -> Создать счет -> Многоразовый -> USDT (0.1$) минимум (Лучше поставить одну валюту USDT дабы не собирать у себя разные токены, конвертировать в USDT будет неудобно и затратно, а и все выплаты в USDT происходят)
4) В настройках счета жмем "Разрешения" - Комментарии ВКЛ, Анонимные платежи ВЫКЛ
5) В настройках счета жмем "Уведомления" -> Подключить -> Добавляем CryptoBot администратором
6) photos/Wins.png - фото при победе, photos/Lose.jpg - фото при проигрыше. Чтобы заменить на свой баннер, просто вставьте свой баннер с таким же названием и замените свой на текущий


Общие настройки:
Созданного бота добавляете администратором в:
1) Игровой канал
2) Канал выплат
3) Канал логов

Получаем id чатов итд с помощью @username_to_id_bot
Ставим на сервере/дедике московский часовой пояс (Удаление чеков в 00:00 по таймеру)
